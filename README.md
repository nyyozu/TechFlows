# TechFlows

O projeto, intitulado "TechFlows Solutions", terá como escopo inicial a implementação de uma tela de autenticação de usuário.

Funcionalidades Principais:

Uma página web (login.html) contendo um formulário com os campos "Usuário" e "Senha", e um botão "Entrar".

Um backend (Python/Flask) que recebe os dados do formulário.

Uma lógica de validação simples que verifica se o usuário e a senha correspondem a valores pré-definidos (ex: usuario="admin" e senha="1234").

Feedback visual para o usuário, informando se o login foi bem-sucedido ou se falhou.
